# **App Name**: GastroFlow

## Core Features:

- Real-Time Dashboard: Displays a dashboard summarizing key restaurant metrics such as orders, sales, and inventory levels.
- Dynamic Menu Management: Enables staff to manage the menu, including adding, editing, and deleting food items, categories, and prices.
- Streamlined Order Handling: Allows the creation and management of orders for dine-in, takeaway, and delivery, assigning them to tables or delivery staff.
- Efficient Inventory Control: Simplifies inventory tracking with features to monitor stock levels and manage vendor information.
- AI-Powered Special Suggestion: Generates potential daily specials based on available ingredients and current inventory using AI. The tool will check inventory levels to suggest profitable and practical dishes, helping minimize waste and maximize profits.
- Automated Staff Scheduling: Facilitates the scheduling of staff shifts and tracks attendance, improving workforce management.

## Style Guidelines:

- Primary color: Rich coral (#FF7F50) to evoke warmth and appetite.
- Background color: Soft peach (#FAEBD7) for a comforting, neutral backdrop.
- Accent color: Mustard yellow (#FFDB58) for highlights and calls to action.
- Headline font: 'Belleza' serif for a sophisticated, fashionable appeal.
- Body font: 'Alegreya' serif for readability in longer texts. Note: 'Belleza' is for headlines and 'Alegreya' is for body.
- Minimalist icons representing menu categories and dashboard functions.
- Clean and spacious layout with clear section divisions to enhance user navigation.
